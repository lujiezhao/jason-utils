## 简介

个人前端项目开发工具函数抽离，维护工具函数包！

## 安装使用

### 1. 安装包使用方式

安装

```shell
# npm
npm install lmkj-utils -S

# yarn
yarn add lmkj-utils -S
```

ESM 导入

```js
import { random } from 'lmkj-utils'
console.log(random(1, 10))
```

RequireJS 导入

```js
const { random } = require('lmkj-utils')
console.log(random(1, 10))
```
